- Title: "VoiceOver Setup Guide - WCAG 2.2 Demo"
- Meta description: "Built into macOS and iOS. System-level screen reader for testing."
- Hero eyebrow: "Setup Guide", title: "VoiceOver Screen Reader (macOS)"
- Content sections (h3):
  1. **Enable VoiceOver on macOS** — System Preferences → Accessibility → VoiceOver, click "Enable" (or use Cmd+F5 hotkey)
  2. **First-Time Activation** — VoiceOver will speak a startup message; if audio is not working, check System Preferences → Sound
  3. **Learn VoiceOver Navigation** — VO (Control+Option) + arrow keys to navigate; VO+Space to activate; VO+H for help menu
  4. **Test with Checkpoint Demo** — Open a checkpoint page in Safari or Chrome, use VO + arrow keys to explore the demo, listen to announcements
  5. **Focus Indicators** — VoiceOver shows a blue spotlight around focused elements; press VO+F3 to show the rotor (menu of headings, links, etc.)
  6. **Built-in, No Installation** — VoiceOver is included with macOS; no download or install needed
- Note: macOS-specific, not available on Windows; JAWS is Windows alternative
  </action>
