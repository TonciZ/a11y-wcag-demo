# WCAG 2.2 Demo Site

Interactive accessibility demos showing WCAG failures and how to fix them. Built for testers and developers.

**Live site:** https://tonciz.github.io/a11y-wcag-demo/

## What it is

Each checkpoint includes:
- A toggleable FAIL / PASS demo
- Explanation of why the failure occurs
- Manual testing instructions
